# NBAStats

A [Limnoria](https://github.com/ProgVal/Limnoria) plugin to retrieve NBA stats from NBA.com.

## Requirements
* Python 3
* [Requests](http://docs.python-requests.org/en/master/)
* [CacheControl](https://cachecontrol.readthedocs.io/en/latest/)
